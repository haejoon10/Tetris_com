# 🎮 BGM 테트리스

브라우저에서 바로 즐길 수 있는 **HTML/JS 기반 테트리스 게임**입니다.  
특징:
- 다양한 블록 모양
- 블록이 아래에 정확히 깔림
- 한 줄 제거 시 배경색 랜덤 변경 및 화면 반짝임
- 블록이 떨어지는 속도 점점 증가
- 점수 및 레벨 표시
- 배경음악(BGM) 지원

---

## 🕹️ 게임 플레이

- **←** : 왼쪽 이동  
- **→** : 오른쪽 이동  
- **↑** : 회전  
- **↓** : 빠르게 떨어뜨리기  
- **🎵 음악 버튼** : 배경음악 켜기/끄기

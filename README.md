# 🎮 모바일 지원 테트리스

브라우저에서 즐길 수 있는 **HTML/JS 기반 테트리스 게임**입니다.  
모바일과 데스크톱 모두 지원하며, 배경음악(BGM), 점수, 레벨 표시, 속도 증가, 배경색 효과 등 다양한 기능을 제공합니다.

---

## 🌟 특징

- 다양한 블록 모양
- 블록이 정확히 바닥에 깔림
- 한 줄 제거 시 **배경색 랜덤 변경** 및 **화면 반짝임**
- 블록이 떨어지는 속도 점점 빨라짐
- 점수 및 레벨 표시
- **배경음악(BGM) 지원**
- 모바일 터치 지원: **좌/우 드래그로 블록 이동**
- 모바일 회전 버튼으로 블록 회전

---

## 🕹️ 게임 조작

### 데스크톱
- **← / →** : 블록 이동  
- **↑** : 블록 회전  
- **↓** : 블록 빠르게 내리기  
- **🎵 음악 버튼** : 배경음악 켜기/끄기  

### 모바일
- **좌/우 드래그** : 블록 이동  
- **터치 후 떼기** : 블록 빠르게 내리기  
- **회전 버튼 클릭** : 블록 회전  
- **🎵 음악 버튼** : 배경음악 켜기/끄기
